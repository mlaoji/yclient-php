# YClient with PHP for YGO

php 版的YClient, 用于访问YGO提供的grpc服务

vendor 下依赖的相关包文件是被我删减过的轻量版,请放心使用

## Installation
-------
 - Install the gRPC PHP extension

首先需要安装PHP grpc 扩展

   ```sh
   $ [sudo] pecl install grpc
   ```
   使用pecl安装或你擅长的方式安装


TRY IT!
-------

 - Run YGO server

启动一个由YGO创建的服务

 - Edit example.php and run it!

编辑并执行example.php
